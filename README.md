# Real-Time Multi-Lane Embedded Racing Game

Embedded Systems Project developed on PIC16F877A using PIC Assembly Language.

Features:
- KS0108 128x64 GLCD
- Three-lane racing gameplay
- Collision detection
- Fuel pickup system
- EEPROM high score storage
- Finite State Machine architecture
- Timer0 interrupt-driven execution

Tools:
- MPLAB X IDE
- pic-as
- SimulIDE
